DESCRIPTION
-----------
This distribtion include all necessary source code files to perform the
following operations on T89C51CC01 product with a CAN bootloader:
         Read and write bytes in the Flash memory
         Read and write the Security Bits
         Read the Software Boot Vector and Boot Status Byte
         Erase and write the Software Boot Vector and Boot Status Byte
         Read the Manufacturer ID, Device ID and bootloader version
         Read and write bytes in the EEPROM
	- Startbootloader

More details about functionnal description and intallation can be found in the
INSTALL file and the application note :
"In-System Programming - Flash Libary for T89C51CC01 with CAN bootloader"

This distribution also includes a simple "example.c" to show the
functionnality of the Flash_API library.
